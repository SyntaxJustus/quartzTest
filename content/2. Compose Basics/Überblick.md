

>[!abstract] Compose
>Jetpack Compose ist ein deklaratives Framework zur Erstellung von Android Apps.
Die UI besteht aus Komponenten, die ***Composables*** genannt werden.
Dabei können vorgegebene Elemente oder selbst erstellte Komponenten genutzt werden.
>
> >[!example] Beispiel
> >Composables sind Kotlin-Funktionen und müssen mit der @Composable Annotation versehen werden.
> >```kotlin
> >@Composable
> >fun MeineKomponente(
> >// Hier werden Parameter festgelegt
> >) {
> >// Inhalt der Komponente
> >}
> >```
>
> >[!info]- Konventionen
> >Entgegen der normalen Kotlin [[Konventionen]] werden Komponentennamen groß geschrieben.
> >Typischerweise wird für jede Komponente eine eigene Datei angelegt.
